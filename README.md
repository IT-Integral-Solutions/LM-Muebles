## Tech Stack

- **HTML5** — Semantic, single-file structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Intersection Observer API, animated counters, custom cursor, scroll effects
- **Google Fonts** — Cormorant Garamond (serif display) + Syne (sans-serif UI)

No frameworks. No build tools. No dependencies.

## Design Details

- Dark luxury aesthetic with gold (`#C8933A`) accent palette
- Custom animated cursor with magnetic ring effect
- Noise texture overlay for depth
- Scroll-triggered reveal animations via Intersection Observer
- Animated stat counters on viewport entry
- Animated roadmap timeline fill
- Sticky navigation with blur backdrop on scroll
- Infinite-loop horizontal ticker
- Fully responsive layout
